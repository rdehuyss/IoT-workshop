# IoT workshop

This is the Github Repository for the IOT Workshop.

## Prerequisites
### Hardware
- ESP32 dev board running micropython and microusb cable
- sensors (DHT11, DHT22, BME680)
- Breadboard
- Jumper cables

### Software
- Latest Docker and Docker Compose
- PyCharm
- Python 3.6
- Esptool (pip3 install esptool)
- Putty
